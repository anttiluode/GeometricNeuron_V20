Geometric Neuron Adder

An 8-bit ripple-carry adder built from nothing but interference amplitude. Each gate is one resonator: bits are encoded as phasors (0 → +1, 1 → -1), the "soma" sums them, and the output bit is read off an amplitude band on |soma| — exactly the mechanism in resonator_neuron.py / gates.py / adder.py from the GeometricNeuron_V20 repo, ported line-for-line.

Gate weights/bias/band exactly match the source: XOR w=[1,1] b=0 band=[-0.1,1.0], AND w=[1,1] b=-2 band=[3.0,99], OR w=[1,1] b=+2 band=[-0.1,3.0]. Full adder = XOR(a,b)->axb, XOR(axb,cin)->sum, AND(a,b)->ab, AND(cin,axb)->cab, OR(ab,cab)->cout. The soma trace animates the same exponential relaxation as settle_dynamical() (envelope already at drive, soma rings up to the fixed point) — not a generic tween.
